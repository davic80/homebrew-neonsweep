# homebrew-neonsweep

[Homebrew](https://brew.sh) tap for [NeonSweep](https://github.com/davic80/neonsweep), a retro neon-terminal Mac cleaner.

## Install

```sh
brew tap davic80/neonsweep
brew install --cask neonsweep
```

## Update

```sh
brew update
brew upgrade --cask neonsweep
```

See the [main repo](https://github.com/davic80/neonsweep) for what NeonSweep does, the changelog, and how to build it from source.
